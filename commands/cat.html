The cat command

Description:

The cat command is short for concatenate and it is used for concatenating files and displaying the content of a text file.

Usage:

cat [option] [file name]

Examples:

Display the contents of a file

cat file.txt

Concatenate several files

cat ~/Documents/file1.txt ~/Downloads/file2.txt

Display the contents of a file with line numbers

cat -n /proc/meminfo

Create a file

cat > newfile.txt

Display the content of a file with $ to indicate the end of each line

cat -E /etc/passwd
