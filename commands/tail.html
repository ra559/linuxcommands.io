The tail command

Description:

Displays the last 10 lines of a given file

Usage:

tail [option] [file]

Example:

Print the last 10 lines of a file

tail /etc/passwd

Prints the last n number of a file

tail -5 /etc/passwd

Prints the n number of given bytes of a file

tail -c 100 /etc/passwd

Supress the filename when more than one file is given

tail -q /etc/passdwd file.txt

Display appended data

tail -f access.log
