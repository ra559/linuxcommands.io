The gnome-open command

Description:

The gnome-open command opens any file with the dedicated application. The gnome-open command is available with the Gnome Desktop Environment.

Usage:

gnome-open file

Example:

Open a file

gnome-open file.pdf

Open a file manager in a given directory

gnome-open ~/Downloads

Open a file with sudo privileges

sudo gnome-open file.txt
