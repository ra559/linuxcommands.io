The more command

Description:

More is a pager program used for loading the output of a large program or a text file one page at the time. More loads the entire text to the buffer and allows you to move one page at a time.

Usage:

more [option] [file or command output]

Examples:

load a text file

more program.py

Pass the output of a command

lshw | more

Open multiple files (use :n for the next file or :p for the previous file)

more file*

IOpen a file at the first of the occurance a string

more +/^system file.txt

Open a file at a given line number

more +50 access.log

See the man page for more options and a help guide for more
