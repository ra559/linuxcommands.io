The xdg-open command

Description:

The xdg-open command opens any file with the dedicated application. The xdg-open command is available with the XFCE Desktop Environment.

Usage:

xdg-open file

Example:

Open a file

xdg-open file.pdf

Open a file manager in a given directory

xdg-open ~/Downloads

Open a file with sudo privileges

sudo xdg-open file.txt

For more information, read the man page
