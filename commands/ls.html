Using the ls command

Description

list files and directories.

Usage

ls [options] [directory]

Examples:

list all files inside the present working directory

ls

list all files inside the present working directory including hidden files

ls -a

long list all files inside the pwd

ls -l

you can convine different options for example: long list all files including hidden files inside the

ls -la

lets see more complex examples:

Bob wants to list all the files in the Music directory with the author

ls -la --author /home/$USER/Music

Bob's manager asked him to list of all the files (excluding hidden) inside the home directory of a user named Maria including their file size in human readable format and sorted by file size

ls -shS /home/Maria/Downloads

Bob wants to see the size, author, and inode number of 3 directories 'Virtual Machines' Music ".pccc's files"

ls -slid --author "Virtual Machines" Music ".pccc's files"

for more options, read the man page of the ls command
