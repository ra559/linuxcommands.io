Using Wildcards

Description:

Wildcard represents letters and characters used to specify a filename for searches.

File globbing is the processing of pattern matching using wildcards.

The wildcards are officially called metacharacter wildcards.

For Example:

  • You can use a wildcard to get a long list of all files in the current directory starting with “new.”
  • Use wildcards to navigate to directories faster
  • Move or delete a group of files
  • Locate files based on a portion of their filenames
  • Create files and directory  quicker
Wildcard Description Example
* Matches zero or more characters in a file name ls *.txt
? Matches any one character in a filename ls f?l?.txt
[aei] Matches one of multiple characters in a filename; in this case a,e,i ls f[a,e,i]le.txt
[a-z] Matches one of a range of characters in a filename; in this case any character from a to z ls f[a-z]le.txt
[!m-x] Matches filenames that do not contain a specified range of characters;
in this example filenames that do not contain letters m to x		 ls f[!m-x]le.txt

More advanced wildcards

Although this is not filename expansion, another useful bash shell feature is the capability to expand a list of strings contained within curly brackets.

Examples:

ls -ld /etc/rc{0,1,2,3,4,5,6}.d
mkdir movies{1,3,5}
touch notes_for_{math,english,linux}.txt

You can use [...] for filename expansion, for example:

  • The ability to define a range for example [a-z] or [1-9] or [001-3]. Alternatively, you can use: [:alpha:] or [:digit:].
    • [:alpha:] Alphabetic (case insensitive)
    • [:lower:] Lower case printable characters
    • [:upper:] Upper printable characters
    • [:digit:] Numbers in decimal from 0 to 9
    • [:alnum:] Alphanumeric (all digits and alphabetic characters)
    • [:space:] White space (space, tabs, and new lines)
    • [:graph:] Printable characters excluding spaces
    • [:print:] Printable characters including spaces
    • [:punct:] Punctuation
    • [:cntrl:] Control characters
    • [:xdigit:] Hexadecimal characters
  • You can also specify a list of characters to match by placing them between the brackets. Like [aeiou] or [12345]
  • You can also use the [] to match the - character like this [-aeiou] or [aeiou-]
