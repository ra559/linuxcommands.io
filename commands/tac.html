The tac command

Description:

The tac command is used for concatinating and displaying the content of a file in reverse order

Usage:

tac [option] [file name]

Examples:

Display the content of a file in reverse order

tac /etc/passwd

Separate the contents of a file using a string inside the file

tac -s file.txt “ip”

Concatenate several files

tac ~/Documents/file1.txt ~/Downloads/file2.txt
