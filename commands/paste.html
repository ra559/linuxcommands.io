The paste command

Description:

The paste command is used to joing files horizontally in columns.

Usage:

paste [option] [files]

Example:

Merge files with paste

paste users.txt ip.txt

Merge files using a different delimeter

paste -d ">"users.txt ip.txt

Merge files using more than one delimeter

paste -d ":-" users.txt ip.txt country.txt

For more information, read the man page
