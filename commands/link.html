Hard and Soft Links

Hard Links

  • Are files that point to data on the hard drive (associates a name with a file on a file system).
  • Hard links share the same inode number
  • The data isnâ€™t deleted until the last link is deleted
  • If one file is changed, all the hard links will change as well
  • To create a hard link use the command:

    Usage

    ln + file + new hard link

    Example:

    ln homework.docx homeworkHL.docx

Symbolic (soft) links

  • Are special types of files that point to other files instead of pointing to data on the hard drive.
  • Symbolic links are the equivalent of shortcuts in a Windows environment
  • To create a hard link use the command:

    Usage

    ln -s + file + new hard link

    Example:

    ln -s homework.docx homeworkSL.docx
