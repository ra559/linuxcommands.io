The head command

Description:

Displays a given number of lines or bytes of a file. If no option given it displays the fist 10 lines of a file

Usage:

head [option] [file]

Example:

Print the first 10 lines of a file

head /etc/passwd

Prints the first n number of a file

head -5 /etc/passwd

Prints the n number of given bytes of a file

head -c 100 /etc/passwd

Supress the filename when more than one file is given

head -q /etc/passdwd file.txt

For more information, read the man page
