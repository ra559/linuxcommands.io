The more command

Description:

Less is a pager programm that allows you to load the output of a command or a file 1 page at a time to the bufffer. Different than more, less is faster when working with large files because it loads the output to the buffer a page at a time.

Usage:

more [option] [file or command output]

Examples:

load a text file

less program.py

See the man page for more options and a help guide for less
